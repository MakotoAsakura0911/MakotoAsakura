input type="reset"

<input type="reset" value="リセット">を使用するとリセットボタンを作成することができます。

リセットボタンは、入力フォームに記入した内容を全て削除し、入力する前の状態に戻す（リセットする）ボタンです。

value属性値に記述された文字（上記の例では、「リセット」）が実際にボタンとして表示される文字です。

<input>タグのtype属性値に「button」を記述した場合、フォーム用以外の様々なボタンを自由に作成することができます。　ただし、JavaScript等と合わせて使用する必要があります。

属性

name属性：ボタンに名前をつける

value属性：ボタンの値を指定する 

						<form>
						名前：<input type="text" name="名前"><br>
						パスワード：<input type="password" name="パスワード"><br>
						内容：<textarea name="内容"></textarea><br><br>
						<input type="submit" value="送信">
						<input type="reset" value="リセット">
						</form>
