input type="file"

<input>タグのtype属性でtype="file"を指定すると、 ファイル名の入力フィールドと[参照]ボタンが表示され、サーバーへ送信するファイルを選択できるようになります。 サーバー側にはファイルを受け取るためのプログラムが必要となります。

<input>タグにtype="file"を指定する場合、 <form>タグにはmethod="post"、および、enctype=" multipart/form-data"を指定するようにしてください。

属性

name属性：フォーム部品に名前をつける

size属性：フォーム部品のサイズを指定する

accept属性：type="file"の場合に、どのタイプのファイルがサーバ側で受け取ることができるかを指定

multiple属性：複数の値を入力・選択できるようにする

required属性：選択必須にする 

						<form action="cgi-bin/abc.cgi" method="post" enctype="multipart/form-data">

						<input type="file" name="datafile">
						</p>
